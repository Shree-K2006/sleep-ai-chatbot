# 🧠 AI Powered Sleep Cycle Improvement Chatbot

An AI-powered chatbot designed to help users improve their sleep cycle and maintain better mental health.

## 🚀 Features
- User Signup and Login
- AI Chatbot for sleep guidance
- Sleep cycle tracking
- Light/Dark theme toggle
- Age-based sleep suggestions

## 🛠 Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenRouter / Gemini API

## ▶️ How to Run the Project

1. Clone the repository
git clone https://github.com/Shree-K2006/ai-sleep-chatbot.git

2. Install dependencies
pip install flask requests

3. Run the project
python app.py

4. Open browser
http://127.0.0.1:5000
